# 🛍️ 电商作图使用指南

## 📋 概述

本指南介绍如何使用小巨蛋智能体的电商作图功能，通过飞书机器人快速生成符合各电商平台规范的图片。

## 🚀 快速开始

### 第一步：环境准备
```bash
# 1. 给启动脚本执行权限
chmod +x start_ecommerce_bot.sh

# 2. 查看功能列表
./start_ecommerce_bot.sh features

# 3. 安装和配置
./start_ecommerce_bot.sh install
```

### 第二步：飞书配置
1. 访问 [飞书开放平台](https://open.feishu.cn/app)
2. 创建企业自建应用
3. 开启机器人能力
4. 获取以下信息：
   - **App ID**
   - **App Secret** 
   - **Verification Token**

### 第三步：启动机器人
```bash
./start_ecommerce_bot.sh start
```

## 📸 核心功能

### 1. 白底图生成
**适用平台**：淘宝、天猫、京东等

**规格要求**：
- 尺寸：800×800px
- 背景：纯白色 (#FFFFFF)
- 格式：JPG
- 要求：无logo、无水印、无阴影

**使用方法**：
在飞书群中发送：
```
生成淘宝白底图
```
或
```
白底图帮助
```

### 2. 商品详情图
**包含内容**：
1. 主展示图 - 产品整体
2. 细节展示图 - 材质工艺
3. 使用场景图 - 实际场景
4. 参数规格图 - 技术参数
5. 对比展示图 - 竞品对比

**使用方法**：
```
制作手机详情图
```
或
```
详情图模板
```

### 3. 透明底图
**适用平台**：抖音、海报设计、广告素材

**规格要求**：
- 比例：9:16（抖音）
- 格式：PNG
- 背景：透明
- 质量：高清抠图

**使用方法**：
```
生成抖音透明底图
```

### 4. 营销图片
**适用场景**：促销活动、社交媒体、广告投放

**包含元素**：
- 价格标签
- 促销文案
- 倒计时
- 优惠券信息

**使用方法**：
```
制作双11促销图
```

## 📋 平台规范指南

### 淘宝/天猫
```
平台规范 淘宝
```
**输出**：
- 主图：800×800px，JPG，白底
- 详情图：宽度750px+，JPG
- 要求：无logo、无水印、清晰

### 京东
```
平台规范 京东
```
**输出**：
- 主图：800×800px，可带品牌
- 详情图：790px宽度
- 建议：参数展示、认证标识

### 抖音
```
平台规范 抖音
```
**输出**：
- 比例：9:16 竖版
- 格式：PNG（透明底）
- 适配：短视频封面、直播预告

### 拼多多
```
平台规范 拼多多
```
**输出**：
- 尺寸：400×400px+
- 特点：价格突出、团购信息
- 显示：销量、评价

### 小红书
```
平台规范 小红书
```
**输出**：
- 比例：3:4 或 1:1
- 风格：生活方式、使用教程
- 特点：用户分享、对比评测

## 💡 最佳实践

### 图片准备建议
1. **高质量原图**
   - 分辨率：至少2000×2000px
   - 光线：均匀自然光
   - 背景：简单纯色

2. **产品展示**
   - 多角度拍摄
   - 展示全部细节
   - 保持产品完整

3. **格式选择**
   - 白底图：JPG（高质量）
   - 透明底：PNG-24
   - 详情图：JPG（压缩优化）

### 文案优化建议
1. **核心卖点**
   - 突出产品优势
   - 简洁明了
   - 符合平台调性

2. **营销话术**
   - 促销活动文案
   - 限时优惠强调
   - 行动号召

3. **平台适配**
   - 淘宝：专业正式
   - 抖音：活泼有趣
   - 小红书：真实分享

## 🔧 故障排除

### 常见问题

#### 1. 机器人不响应
- ✅ 检查飞书事件订阅配置
- ✅ 验证Verification Token
- ✅ 确认机器人已加入群聊

#### 2. 图片生成失败
- ✅ 检查原图质量
- ✅ 确认网络连接
- ✅ 查看日志文件

#### 3. 平台审核不通过
- ✅ 检查尺寸规范
- ✅ 优化图片清晰度
- ✅ 避免违规内容

```bash
# 实时查看日志
tail -f logs/ecommerce_bot.log
```
