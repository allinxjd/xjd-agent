# 技能：卡拉贝斯AI作图功能详解

## 技能名称
`calabash_ai_drawing`

## 功能概述
卡拉贝斯AI平台提供专业的电商图片生成服务，结合AI模型智能生成符合各平台规范的营销图片。

## 核心作图功能

### 1. 产品图片生成
#### 1.1 产品单图
- **功能**：生成单个产品的展示图片
- **适用**：产品列表、商品卡片
- **特性**：
  - 支持透明背景
  - 自动产品抠图
  - 智能光影处理
  - 多角度展示

#### 1.2 产品主图
- **功能**：生成电商平台商品主图
- **适用**：淘宝、京东、拼多多等
- **特性**：
  - 平台规格适配
  - 营销文案集成
  - 卖点突出展示
  - 品牌元素融合

#### 1.3 产品详情图
- **功能**：生成商品详情页图片
- **适用**：商品详情描述
- **特性**：
  - 多图组合
  - 功能展示
  - 使用场景
  - 规格参数

### 2. 营销图片生成
#### 2.1 促销广告图
- **功能**：生成促销活动图片
- **特性**：
  - 价格标签
  - 促销文案
  - 倒计时
  - 优惠信息

#### 2.2 社交媒体图
- **功能**：生成社交媒体内容
- **适用**：抖音、小红书、微信
- **特性**：
  - 竖版适配
  - 互动元素
  - 话题标签
  - 分享优化

#### 2.3 横幅海报
- **功能**：生成横幅广告
- **适用**：网站横幅、活动海报
- **特性**：
  - 多种尺寸
  - 品牌一致性
  - 视觉冲击力
  - 信息层次

## 平台规格支持

### 抖音平台
- **尺寸**：9:16 竖版
- **格式**：支持透明PNG
- **特性**：
  - 短视频封面优化
  - 直播预告图
  - 商品橱窗图
  - 话题挑战图

### 淘宝/天猫
- **主图**：800×800px
- **详情图**：宽度750px+
- **特性**：
  - 白底图要求
  - 多角度展示
  - 细节放大
  - 场景图

### 京东
- **主图**：800×800px
- **详情图**：790px宽度
- **特性**：
  - 品牌认证
  - 参数展示
  - 对比图
  - 认证标识

### 拼多多
- **主图**：400×400px+
- **特性**：
  - 价格突出
  - 团购信息
  - 用户评价
  - 销量展示

### 小红书
- **笔记图**：3:4 或 1:1
- **特性**：
  - 生活方式展示
  - 使用教程
  - 对比评测
  - 用户分享

- **多图上传**：同时处理多个图片
- **统一风格**：保持品牌一致性
- **批量下载**：打包下载结果

## 最佳实践

### 图片准备
1. **高质量原图**：清晰、光线均匀
2. **简单背景**：便于抠图处理
3. **产品完整**：展示全部细节
4. **格式标准**：PNG或高质量JPG

### 文案优化
1. **简洁明了**：突出核心卖点
2. **符合平台**：使用平台流行语
3. **层次分明**：主标题+副标题
4. **调用行动**：促进用户互动

### 平台适配
1. **尺寸检查**：确保符合规范
2. **格式验证**：平台支持格式
3. **文件大小**：优化加载速度
4. **移动端适配**：小屏显示效果

## 常见问题解决

### 生成质量不佳
1. 更换AI模型尝试
2. 优化输入图片质量
3. 调整生成参数
4. 添加更详细描述

### 平台审核不通过
1. 检查尺寸规范
2. 避免违规内容
3. 优化图片清晰度
4. 符合平台调性

### 处理速度慢
1. 选择快速模型
2. 优化图片大小
3. 避开高峰时段
4. 使用批量处理
