# 技能：卡拉贝斯AI图片生成

## 技能名称
`calabash_upload_generate`

## 功能描述
在卡拉贝斯AI平台上上传图片，选择产品单图功能，生成透明底图片，平台选择抖音。

## 适用场景
- 电商产品图片处理
- 社交媒体内容制作
- 广告素材生成

## 前置条件
1. 已打开浏览器访问 https://ai.calabashai.cn
2. 已登录账户（如需完整功能）
3. 图片文件已准备好

## 操作步骤

### 步骤1：打开平台
```bash
open "https://ai.calabashai.cn"
```

### 步骤2：准备图片文件
- 确认图片文件路径和名称
- 检查文件格式（支持PNG、JPG等）
- 示例：`/Users/evnzhang/Desktop/xjdlogo.PNG`

### 步骤3：上传图片
1. 在平台首页找到上传区域
2. 点击上传按钮或拖拽文件
3. 选择目标图片文件

### 步骤4：选择功能
1. 选择"产品单图"功能
2. 确保选择透明底选项（如有）

### 步骤5：选择平台
1. 在平台选择区域找到"抖音"
2. 选择抖音平台规格

### 步骤6：生成图片
1. 点击生成按钮
2. 等待处理完成
3. 预览生成效果

### 步骤7：下载图片
1. 点击下载按钮
2. 选择保存位置
3. 确认文件格式和质量

## 文件处理命令

### 检查桌面图片
```bash
ls -la ~/Desktop/*.png ~/Desktop/*.PNG ~/Desktop/*.jpg ~/Desktop/*.JPG
```

### 查看图片信息
```bash
file ~/Desktop/xjdlogo.PNG
```

### 打开图片所在目录
```bash
open ~/Desktop
```

## 平台规格要求

### 抖音平台
- 尺寸要求：根据抖音规范
- 格式要求：支持透明PNG
- 文件大小：建议优化

### 产品单图功能
- 生成单一产品展示图
- 支持透明背景
- 自动适配平台规格

## 错误处理

### 常见问题
1. **上传失败**：检查文件格式和大小
2. **生成失败**：检查网络连接，重试
3. **透明底无效**：确认原图有透明通道

### 解决方案
1. 重新上传文件
2. 刷新页面重试
3. 联系平台支持

## 优化建议
1. 批量处理时建立文件列表
2. 保存常用平台配置
3. 定期清理生成历史
